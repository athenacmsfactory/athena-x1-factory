# Specification: Systematic Site Optimization (20260301)

## Overview
This track aims to systematically optimize all sites generated by the Athena 2 factory, focusing on Core Web Vitals (LCP, CLS, TBT), Image Optimization, and CSS/Tailwind delivery. The goal is to achieve Lighthouse scores of 90+ across all categories and meet standard Field Data (CWV) thresholds.

## Functional Requirements
- **Foundation Cleanup:** Perform a comprehensive audit and cleanup of redundant JS and CSS across the base templates.
- **Scripts Optimization:** Implement strategies (defer, lazy loading, Partytown) for third-party scripts (analytics, widgets, etc.) to minimize main-thread impact.
- **Font Loading:** Transition from external font providers to self-hosted or system-stack solutions to reduce render-blocking requests.
- **Image Optimization:** Enhance image handling with lazy loading, modern formats (WebP/AVIF), and optimized hero section preloading for LCP.
- **Systematic Application:** Roll out these optimizations across all existing sites in the `sites/` directory sequentially.

## Non-Functional Requirements
- **Performance:** Target Lighthouse scores of 90+ for Performance, Accessibility, Best Practices, and SEO.
- **Core Web Vitals:** Ensure LCP < 2.5s, CLS < 0.1, and FID/INP within "Good" thresholds.
- **Maintainability:** Ensure optimizations are integrated into the base templates and factory engine for future sites.

## Acceptance Criteria
- [ ] Baseline performance audit performed for a set of pilot sites.
- [ ] Foundation cleanup implemented in base templates.
- [ ] Scripts and font loading optimizations verified in production builds.
- [ ] Image optimization logic updated and verified.
- [ ] All sites in the `sites/` directory successfully updated with optimizations.
- [ ] Final Lighthouse reports confirm 90+ scores for the pilot sites.

## Out of Scope
- Integration of automated Lighthouse CI in the `athena-publisher` workflow (deferred for now).
- Significant structural redesigns of the sites (focus is on technical optimization).
