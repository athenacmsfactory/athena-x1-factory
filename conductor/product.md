# Initial Concept
Athena 2 is an automated factory for React 19 + Tailwind v4 sites, built as a monorepo. It bridges the gap between high-end developer-driven performance and client-friendly content management.

## Target Audience
- **Web Developers:** Seeking to automate boilerplate, deployment, and management of high-performance React sites.
- **Small Business Owners/Clients:** Individuals who need modern, premium websites that they can easily update via Google Sheets without coding knowledge.
- **Agencies:** Scalable "site factory" solution for producing and managing a large volume of client sites efficiently.

## Primary Goals
- **Extreme Automation:** Automating the entire lifecycle from content ingestion (Google Sheets) to deployment via GitHub Actions.
- **Visual Empowerment:** Providing a real-time visual editor (Athena Dock) for intuitive, code-free site customization.
- **Infrastructure Efficiency:** Maintaining a lightweight, high-performance monorepo architecture that handles dozens of sites with minimal overhead.

## Key Features
- **Google Sheets Sync:** Real-time synchronization between Google Sheets (as a headless CMS) and the React frontend.
- **Visual Editor (Dock):** A sophisticated `postMessage` based visual editor for non-technical users to edit styles and layout.
- **Multi-Provider AI:** A robust AI waterfall engine (supporting Google, Groq, OpenRouter, etc.) for automated content and asset generation.
- **Blueprint/Sitetype System:** Reusable business-specific blueprints for rapid, consistent site generation across different industries.

## Design Philosophy
- **Premium & Minimalist:** Modern aesthetics characterized by high-quality imagery, generous whitespace, and professional typography.
- **Performance-First:** Optimized for lightning-fast load times and high Core Web Vitals using React 19, Vite, and Tailwind v4.
- **Highly Adaptable:** Architecture designed for easy skinning and customization via global variables and sitetype-specific configurations.

## Core Mission
- **Democratizing High-End Web Design:** Making premium, high-performance websites accessible to small businesses without the "WordPress bloat."
- **Developer Productivity:** Eliminating repetitive tasks, allowing developers to focus on unique custom logic rather than boilerplate.
- **Content-First Independence:** Empowering non-technical users to manage their data through familiar tools like Google Sheets, ensuring independence from code for daily updates.
